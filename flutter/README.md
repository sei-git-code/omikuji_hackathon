# おみくじアプリ (Flutter)

## セットアップ

```bash
cd flutter
flutter pub get
```

## 起動方法

### iOS Simulator / Android Emulator
```bash
flutter run
```

### 特定のデバイス
```bash
flutter devices
flutter run -d <device-id>
```

## 機能

- おみくじを引くボタンでランダムに結果を表示
- 結果の管理機能（追加・削除）
- シンプルな白背景のデザイン
- **API非依存**: おみくじ結果はアプリ内で管理され、ランダムに選択されます

## おみくじ結果の管理

- デフォルトで以下の7つの結果が設定されています：
  - 大凶
  - 凶
  - 末吉
  - 小吉
  - 中吉
  - 吉
  - 大吉
- 「結果を管理する」ボタンから、結果の追加・削除が可能です
- 結果はアプリ内のメモリで管理されます（アプリを再起動するとデフォルトに戻ります）
