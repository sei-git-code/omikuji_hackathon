# おみくじアプリ (React)

## セットアップ

```bash
cd react
npm install
```

## 起動方法

```bash
npm start
```

アプリは `http://localhost:3000` で起動します。

## 機能

- おみくじを引くボタンでランダムに結果を表示
- 結果の管理機能（追加・削除）
- シンプルな白背景のデザイン

## 環境変数

`.env` ファイルを作成してAPIのURLを設定できます：

```
REACT_APP_API_URL=http://localhost:8080/api/omikuji
```
